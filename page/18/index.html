以下代码创建了一个Point构造器


function Point(x, y){

this.y = y;


}


Point.prototype.move = function(x, y){


this.x += x;


this.y += y;


}

构造器对象相对于普通对象（回顾一下“对象原型链的说明”中的p对象）有如下几个特点：


构造器对象原型链倒数第二个proto是一个Function.prototype对象引用,因此可以调用Function.prototype的属性和方法。


构造器对象本身有一个prototype属性，用该构造器实例化对象时该prototype会被实例对象的proto所引用。


构造器对象本身是一个function对象，因此会有name,length等自身属性。


 


 


JS对象原型链的简要说明


以下代码创建了一个Point构造器并实例化了一个p对象


function Point(x, y){


this.x = x;


this.y = y;


}


Point.prototype.move = function(x, y){


this.x += x;


this.y += y;


}


var p = new Point(1,1);


p.move(1,2);


在调试中展开p对象结构如下图


“proto“就是我们通常说的原型链属性，他有如下几个特点：


“proto“是对象的一个内部隐藏属性。


“proto“是实例化该对象的构造器的prototype对象的一个引用，因此可以访问prototype的所有属性和方法。


除了Object对象，每个对象都有一个”proto“属性，”proto“逐级增长形成一个链就是我们所说的原型链，原型链顶端是一个Object对象。


当开发者调用对象属性或方法时(比如”p.move(1,2)”)，引擎首先会查找p对象的自身属性，如果自身属性中没有”move”方法，则会继续沿原型链逐级向上查找，直到找到该方法并调用。


“proto“跟浏览器引擎实现相关，不同的引擎中名字和实现不尽相同(chrome、firefox中名称是”proto“，并且可以被访问到，IE中无法访问)。基于代码兼容性、可读性等方面的考虑，不建议开发者显式访问”proto“属性或通过”proto“更改原型链上的属性和方法，可以通过更改构造器prototype对象来更改对象的proto属性。

页面制作之盒模型

CSS盒模型


一:盒子模型

-每个盒子都有:边距、边框、填充、内容四个属性

-每个属性都有:上、下、左。右四部分


二:CSS盒模型

-margin border padding content

-content=widthheight

-border,padding,margin都有top right bottom left四部分


三:width

-width:<length>|<precentage>

-引申:max-width,min-width

-width:50%;


四:height

-<length>|<precentage>

-height:50%;

-引申:max-height,min-height


五:padding 填充

-[<length>|<precentage>]{1,4}

-padding:20px 20px 20px 20px

上 右 下 左

-也可以单独设置　padding-top:10px;

-引申:margin,border都有四个值


-值缩写

-padding:20px 10px (上下20px 左右10px)

-padding:20px 10px 30px=padding:20px 10px 30px 10px

-对面相等，后者省略，4面相等，只设一个


六:margin

-[<length>|<precentage>|auto]{1,4}

-引申:margin的TRBL,值缩写

-margin合并：

-相邻的元素外边距会合并,取较大的值

-父元素与第一个/最后一个子元素会合并


七:水平居中

-margin:0 auto;(水平居中)


八:border

-[<border-width>||<border-style>||<border-color>]

-border-width:[<length>]{1,4}

-border-style:[solid|dashed|dotted]{1,4}

-border-color:[<color>|transparent]{1,4}


九:border-radius(圆角)

-top-left,top-right,bottom-right,bottom-left

-[<length>|<precentage>]{1,4}[/<length>|<precentage>]{1,4}?

-border-radius:10px;(四个角都是10px)

-border-radius:0px 5px 10px 15px/20px 15px 10px 5px

top-L水平 -R水平 -L垂直 -R垂直

-border-radius:50%;//正圆

-border-top-left-radius:10px//左上角10px


十:overflow

-visible|hidden|scroll|auto

-overflow:visible//超出部分仍然显示

-overflow:hidden//超出部分隐藏

-overflow:scroll//显示滚动条

-overflow:auto//当元素内容超出显示滚动条

-引申:overflow-x,overflow-y

-content-box|border-box

-none|<shadow>[,shadow]..

-box-shadow: 4px 6px 3px 3px red

水平偏移 垂直偏移 模糊半径 阴影大小

-box-shadow: inset 0px 0px 5px red//内阴影

-box-shadow 3px 3px 5px 2px black,inset 0px 0px 5px red//多阴影

-[<outline-width>||<outline-style>||<outline-color>]

-不占空间
