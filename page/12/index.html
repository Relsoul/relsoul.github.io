零基础学bootstrap:起步

本次课程是零基础开始学bootstrap.因为这个框架比较流行,所以结合自己正在学的视频教程写一篇比较详细的文本型教程出来..文笔比较渣大家勿喷.如果遇到有什么错误请及时留言.我一定会努力改正


PS:这里的零基础指的是bootstrap零基础.要学习bootstrap只需要一定的HTML与CSS基础即可


 


一:bootstrap是什么？


这个问题问得好,bootstrap简单来说就是一个结合了各种黑科技的框架,用了这个框架你不用再考虑手机和电脑自适应的问题,而且最新的bootstrap V3对手机作了很大的优化.bootstrap本身也带了很多炫酷的交互效果特效,而且提供了源码.可以从深层去指定bootstrap.


 


二:如何引用bootstrap?


可以从官方的CDN服务器里引入,也可以从本地引入


本地引入只需要下载bootstrap文件了


http://v3.bootcss.com/getting-started/#download


只需要在头部标签中加入


<link rel=”stylesheet” type=”text/css” href=”../css/bootstrap.min.css”>

<script type=”text/javascript” src=”../js/bootstrap.min.js”></script>


如果你还需要用到bootstrap的 jquery效果 只需要在bootstrap.min.js前引入jquery.js就ok了


PS: .min是压缩后的文件 未加min的是未压缩的文件,两者效果都一样,只是文件大小不一样


 


三:注意事项


当然bootstrap注意事项比较多,大家可以去http://v3.bootcss.com/getting-started/进行详细的查看,这里只稍微说一下比较重要的.首先在一个完整的适配手机版的Bootstrap需要加上一个meta标签的,如下例


<head>
    <meta charset=”utf-8”>
    <meta http-equiv=”X-UA-Compatible” content=”IE=edge”>
    <meta name=”viewport” content=”width=device-width, initial-scale=1”>
    <!– 上述3个meta标签必须放在最前面，任何其他内容都必须跟随其后！ –>

简单解释下 charset=”utf-8” //声明文档类型是utf-8


<meta http-equiv=”X-UA-Compatible” content=”IE=edge”>


IE浏览器中都有兼容模式.为了使浏览器使用最新的兼容模式都要写上这行meta标签


 


最重要的应该是


<meta name=”viewport” content=”width=device-width, initial-scale=1”>


这样做就可以适配移动端了..不要问我为什么要加这条标签.加了就对了.


 


当然你如果要禁用移动端的缩放功能.那么你可以加这一条


<meta name=“viewport” content=“width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no”>


一般来说不推荐禁用移动端的缩放功能.不过谁呢排除特殊情况呢？


bootstrap对浏览器的兼容情况


http://v3.bootcss.com/getting-started/#support


bootstrap是在HTML5下运行的哦.所以请使用HTML5的<!DOCTYPE html>

小号文本


对于不需要强调的inline或block类型的文本，使用 <small> 标签包裹，其内的文本将被设置为父容器字体大小的 85%。标题元素中嵌套的 <small> 元素被设置不同的 font-size 。


你还可以为行内元素赋予 .small 类以代替任何 <small> 元素。


<small>这是bootstrap第三章的排版功能</small>

排版-缩略语


当鼠标悬停在缩写和缩写词上时就会显示完整内容，Bootstrap 实现了对 HTML 的 <abbr> 元素的增强样式。缩略语元素带有 title 属性，外观表现为带有较浅的虚线框，鼠标移至上面时会变成带有“问号”的指针。如想看完整的内容可把鼠标悬停在缩略语上（对使用辅助技术的用户也可见）, 但需要包含 title 属性。



基本缩略语


使用abbr标签,鼠标移入上去会显示title内容


<p>测试文本<abbr title=”缩略词不知道写什么好”>缩略语</abbr></p>

 


首字母缩略语


为缩略语添加 .initialism 类，可以让 font-size 变得稍微小些。


<abbr title=”HyperText Markup Language” class=”initialism”>HTML</abbr>

排版-引用



默认样式的引用


将任何 HTML 元素包裹在 <blockquote> 中即可表现为引用样式。对于直接引用，我们建议用 <p> 标签。


加了一个白色的边框..


<blockquote>
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer posuere erat a ante.</p>
</blockquote>

 


多种引用样式


对于标准样式的 <blockquote>，可以通过几个简单的变体就能改变风格和内容。


命名来源


添加 <footer> 用于标明引用来源。来源的名称可以包裹进 <cite>标签中。



<blockquote>
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer posuere erat a ante.</p>
  <footer>Someone famous in <cite title=”Source Title”>Source Title</cite></footer>
</blockquote>

 



另一种展示风格


通过赋予 .blockquote-reverse 类可以让引用呈现内容右对齐的效果。



<blockquote class=”blockquote-reverse”>
  …
</blockquote>

 




 


排版-列表



无序列表


实现方式以及展示方式跟平时写代码都没什么区别


<ul>
  <li>…</li>
</ul>

 


有序列表


实际上用起来也没太大的差别


<ol>
  <li>…</li>
</ol>

 


无样式列表


移除了默认的 list-style 样式和左侧外边距的一组元素（只针对直接子元素）。这是针对直接子元素的，也就是说，你需要对所有嵌套的列表都添加这个类才能具有同样的样式。


也就是说li下的ul li 不会移除默认的样式


<ul class=”list-unstyled”>
                  <li>我是子li
                    <ul>
                        <li>我是子li的子li</li>
                        <li>我是子li的子li</li>
                    </ul>

                  </li>
                </ul>

 


 


内联列表


通过设置 display: inline-block; 并添加少量的内补（padding），将所有元素放置于同一行。


<ul class=”list-inline”>
                  <li>我是第一个</li>
                  <li>我是第二个</li>
                  <li>我是第三个</li>
                </ul>

 


 


排版-描述



描述


带有描述的短语列表。


<dl>
                  <dt>我是soul</dt>
                  <dd>我是soul的描述-灵魂</dd>
                </dl>

 


水平排列的描述


.dl-horizontal 可以让 <dl> 内的短语及其描述排在一行。开始是像 <dl> 的默认样式堆叠在一起，随着导航条逐渐展开而排列在一行。


在小屏幕下会像普通的dl那样 在屏幕能够支持显示后面的描述后会显示在一行


 


 


<dl class=”dl-horizontal”>
                       <dt>我是渣渣</dt>
                  <dd>我是渣渣的描述-渣渣</dd>
                   <dt>我是soul</dt>
                  <dd>我是soul的描述-灵魂</dd>

                    </dl>

JS程序设计-词法重点

JS-词法重点


一:变量标识符的命名要求



  • 以字母,下划线，或者美元符号($)开头

  • 以字母,下划线,美元符号($)和数组组成



二:关键字与保留字


无法使用关键字与保留字来命名


参考地址:http://www.itxueyuan.org/view/6627.html


三:大小写敏感



  • var Aa=0;与var AA=0;是两个变量

  • 关键字比如for也需要大小写分明


四:严格模式



  • 增加js的安全性

  • 提高编译效率


​


五:使用严格模式



  • 全局使用


<script>

“use strict”

var a=0;

</script>



  • 局部使用


function ab(){

“use strict”

var a=0;

}


六:严格模式与标准模式区别



  • 严格模式下不允许隐式声明


function(){

“use strict”

a=0;//a在前面并未声明 ,使用严格模式会报错


}



  • 对象重命名属性


var obj={a:1,b:2,a:3}

obj.a//非严格模式会得出3 严格模式会报错



  • arguments.callee


var i=0;

function ac(){

if(i>10){

return false;

}

i++;

arguments.callee()//指向本身这个函数,严格模式下会报错

}



  • with语句等等~


七:注释



  • 单行注释


//var a=1;

//var b=2;



  • 多行注释


/var a=1;

var b=2;/

PS切图之测量、取色

 

一:所有数组都要测量.如:宽，高，定位，行高，背景图位置等。

二:利用矩形选框工具结合信息面板进行测量

三:测量时画布尽可能放得够大

四:测量文字大小用文字工具，测量文字行高在文字工具-属性面板可以查到到

五：测量背景图文字-矩形选框工具，测量出的高即为文字大小

六：矩形选框工具


  • 添加到选区:按住shift

  • 从选区减去：按住alt

  • 与选区交叉：按住shift+alt


​


七：小技巧:利用矩形选框工具的添加到选区来测量页面宽/高



  • 在左侧画一个小矩形

  • 在右侧画一个小矩形

  • 信息面板上所得的宽/高即为两者之间的距离


八:取色



  • 尽量用取色器来取色

  • 用取色工具来判断是否为纯色

  • 确定是否为线性渐变:用魔棒工具，把容差改为较低的数值既可以选择到单一的颜色
