CSS布局之两列布局 

<!DOCTYPE html>
<html>
    <head>
        <meta charset=”utf-8”>
        <title></title>
        <style type=”text/css”>
            /一列定宽一列自适应之float+margin
            {padding: 0;margin: 0;}
                .left{
                    float: left;
                    width: 100px;
                    background: red;
                }
                .right{
                    margin-left: 100px; /往右缩进100px; 间距通过margin-left设置
                    background: blue;
                }/

            /*二:一列定宽一列自适应之float+margin+fix
            .left{
                float: left;
                width: 100px;
                position: relative;  提高层级
            }
            .right{
                margin-left: 120px;
            }
            .right-fix{
                float:right;
                width: 100%
                margin-left:-100px;
            }*/


        /*三:一列定宽一列自适应之float+overflow
        .left{
            float:left;
            width: 100px;
            margin-right:20px; /*间距设置在left上
        }
        .right{
            overflow: hidden; /*触发bfc模式,里面内容跟外界隔离,实现所需布局
        }*/

        /*四:一列定宽一列自适应之table
        .parent{
            display: table; 宽度默认为auto
            width: 100%; 设置宽度为100%
            table-layout: fixed; 加速table渲染，实现了布局优先
        }
        .left,.right{
            display:table-cell; left和right变为表格
        }
        .left{
            width:100px;
            padding-right: 20px;设置间距
        }*/


    /*五:一列定宽一列自适应之flex
            .parent{
                display: flex;
            }
            .left{
                width: 100px;
                margin-right:20px ;
            }
            .right{
                flex:1;
            }*/


    &lt;/style&gt;
    &lt;script type="text/javascript"&gt;
        一:一列定宽一列自适应之float+margin
            -优点:容易理解
            -IE6文字会往右缩进3px 通过在left加margin-right可以解决
            -在right子元素上清除浮动.会在right元素上空出left高度的一个空白

        二:一列定宽一列自适应之float+margin+fix
            -不会存在3px问题
            -不会存在right子元素上清除浮动也不会影响
            -right-fix会盖在left上面，所以需要提高left层级

        三:一列定宽一列自适应之float+overflow
            -IE6不支持

        四:一列定宽一列自适应之table
            -代码比较多- -

        五:一列定宽一列自适应之flex
            -不兼容低版本
            -性能会有一个问题(flex布局会涉及到计算)
            -适用于小范围布局


    &lt;/script&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;div class="parent"&gt;
        &lt;div class="left"&gt;
            &lt;p&gt;left&lt;/p&gt;&lt;p&gt;left&lt;/p&gt;
        &lt;/div&gt;
        &lt;div class="right-fix"&gt;
            &lt;div class="right"&gt;
                &lt;p&gt;right&lt;/p&gt;
                &lt;p&gt;right&lt;/p&gt;
            &lt;/div&gt;
        &lt;/div&gt;
    &lt;/div&gt;
&lt;/body&gt;

</html>
