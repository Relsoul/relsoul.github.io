JavaScript:深入浅出-数据类型

学习视频来自:http://www.imooc.com/video/5673


 


 


在JavaScript中数据类型有原始类型和对象


原始类型有以下几类数据



  1. 数字(number)

  2. 字符串(string)

  3. 布尔（boolean)

  4. null

  5. undefined

  6. 对象(object)


对象也算一种原始类型,对象中包含以下几类数据



  1. Function

  2. Array

  3. Date

  4. ….


 


来写一个基本的数据类型


        var num=32;
    var num1=”33”;
    console.log(typeof (num+num1))
    //两个字符串相加是3332并且是字符串类型的
    //字符串相加的时候 如果一方不是字符串会转化为字符串再进行相加
    console.log(typeof (num1*num));
    //相乘则是数字类型
    //字符串相乘的时候 如果一方不是数字会转化Number()为数字再进行相乘

    //函数,数组,日期都是object
