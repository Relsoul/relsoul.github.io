JS程序设计之闭包

闭包

function add(){

var i=0

return function(){

alert(i++)

}

}

var f=add();

f();

f();


-闭包由函数和与其相关引用环境的组合而成

-闭包允许函数访问其引用环境中的变量（又称自由变量）

-广义上来说，所有的JS函数都可以称为闭包，因为JS函数在创建时保存了当前的词法环境
