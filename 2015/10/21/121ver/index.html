JavaScript+css:按钮确认删除效果概念设计

来自：http://www.gbtags.com/gb/share/5151.htm


演示地址:http://www.old.com/demo/%E6%8C%89%E9%92%AE%E8%B7%B3%E8%BD%AC.html


 



分析


首先x√使用了字体图标 详情请参照:http://www.bootcss.com/p/font-awesome/


其次使用了CSS3动画,


设置div的左移动,右移动来实现效果


jquery方面


设置一个定时器 3秒后隐藏√div


 


 


代码


<!doctype html>
<html lang=”en”>
<head>
    <meta charset=”UTF-8”>
    <title>Document</title>
    <style type=”text/css”>
    @import url(‘http://cdn.gbtags.com/font-awesome/4.1.0/css/font-awesome.min.css‘);
     {
      margin: 0;
      padding: 0;
      box-sizing: inherit;
    }

    :focus {
      outline: none;
    }

    html {
      box-sizing: border-box;
    }

    body {
      background-color: #ecf0f1;
      font-family: ‘microsoft yahei’,Arial,sans-serif;
    }
    .button{
        margin: 100px;
        position: relative;
        display: inline-block;
        width: 200px;
        height: 50px;
        background:#C0392B;
        border-radius: 5px;
        overflow: hidden;
    }
    .button span{
        position: absolute;
        text-align: center;
        font-size: 20px;
        line-height: 50px;
        top:0;
        left: 25px;
        color: #fff;
        width: 130px;
        border-right:1px solid #A53125;
        transition: all 0.5s ease;
        }
    .fa-check{
        position: absolute;
        color: #fff;
        text-align: center;
        font-size: 26px;
        line-height: 50px;
        top: 0;
        width: 200px;
        left: 0px;
        background: #27AE60;
        transition: all 0.5s ease;
        display: none;
        border-radius: 5px;
    }

    .fa-times{
        position: absolute;
        color: #fff;
        text-align: center;
        font-size: 26px;
        line-height: 50px;
        top: 0;
        width: 200px;
        left: 75px;
        transition: all 0.5s ease;
        border-radius: 5px;
    }

    .button:hover span{
        transform:translate(-160px,0px);
        border-radius: 5px;

    }

    .button:hover .fa-times{
        transform:translate(-75px,0px);
        background:#C44C3F;
        border-radius: 5px;
    }

    </style>
</head>
<body>
    <a class=”button” role=”button”>
        <span>删除</span>
        <div class=”icon”>
            <i class=”fa fa-times”></i>
            <i class=”fa fa-check”></i>
        </div>
</a>

<!– 推荐开源CDN来选取需引用的外部JS //–>
<script type=”text/javascript” src=”jquery-1.9.1.min.js”></script>
<script type=”text/javascript”>
    $(function(){
        var time;
        var num=0;
        $(“.button”).click(function(){
            var show=function(){
                    $(“.fa-check”).hide(300)
                    time=null;

                }

            if(!time){
                $(“.fa-check”).show()
                time =setTimeout(show,3000)
            }


        })

    })


</script>

</body>
</html>
