零基础学bootstrap:代码

前文


这里也只列出一些比较重点的内容,详细的可以去官方参考文档 http://v3.bootcss.com/css/#code


 


<style>
    .row{
        margin-top: 10px;
        background: #b3b3b3;
        padding: 10px;
    }
    [class*=’col-‘]{
        border:1px solid #2b2f2f;
        padding: 10px;
    }


    </style>

稍微加一点css样式方便区别


 


 


 


内联代码


通过 <code> 标签包裹内联样式的代码片段。 显示是颜色变化.实践一下就知道了


<div class=”container”>
        <div class=”row”>
            <div class=”col-md-8”>
                <p>bootstrap零基础学习-排版,<code>By-Soul</code></p>

            </div>

        </div>

    </div>


 


用户输入


通过 <kbd> 标签标记用户通过键盘输入的内容。..实际上这只是一个样式罢了不会实际上记录你的键盘


显示样式大家可以实践下


<p>我输入了<kbd>ctrl+m</kbd></p>


 


代码块


多行代码可以使用 <pre> 标签。为了正确的展示代码，注意将尖括号做转义处理。


<pre>&lt;p&gt;Sample text here…&lt;/p&gt;</pre>

 


 


变量


通过 <var> 标签标记变量。实际上这只是一个样式罢了


<var>y</var> = <var>m</var><var>x</var> + <var>b</var>


 


程序输出


通过 <samp> 标签来标记程序输出的内容。


展示效果跟普通的差不多.


<samp>This text is meant to be treated as sample output from a computer program.</samp>
