CSS网页自适应布局之单列,双列,三列,混合布局,

首先来说单列吧


单列就是整个页面只有一竖排,实现的方式很简单,看代码即可


单列布局只单纯的用到了 margin:0 auto 实现页面居中的方式


<!DOCTYPE html PUBLIC “-//W3C//DTD XHTML 1.0 Transitional//EN” “http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd"&gt;
<html xmlns=”http://www.w3.org/1999/xhtml"&gt;
<head>
<meta http-equiv=”Content-Type” content=”text/html; charset=utf-8” />
<title>一列布局</title>
<style type=”text/css”>
body{ margin:0; padding:0; font-size:30px}
div{ text-align:center; font-weight:bold}
.main,.footer{ width:960px; margin: 0 auto;}
.head{ width:100%; height:100px; background:#ccc;}
.main{ height:600px; background:#FCC; ;}
.footer{ height:50px; background:#9CF;}
</style>
</head>

<body>
<div class=”head”>head</div>
<div class=”main”>main</div>
<div class=”footer”>footer</div>
</body>
</html>

 


然后是双列布局


还是先看代码


显示一个div盒子包裹者两个div 并且把父div设置页面居中,然后再把里面的两个子div设置浮动既可以实现双列布局


 


<!DOCTYPE html PUBLIC “-//W3C//DTD XHTML 1.0 Transitional//EN” “http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd"&gt;

div{ text-align:center; line-height:50px}
.main{ width:960px; height:600px; margin:0 auto}
.left{ width:300px; height:600px; background:#ccc; float:left;}}/左浮动样式/
.right{ width:660px; height:600px; background:#FCC;float:right;}/右浮动样式/
</style>
</head>

<div class=”main”>
    <div class=”left”>left</div>
    <div class=”right”>right</div>
</div>
</body>
</html>

 


最后是三列布局，三列布局用到了position定位


三列布局先用一个父的div:par包裹起来 并且把父的div设置position为relative


然后再把left,right设置position为absolute;


并且设置left的定位为父div的左上角，right定位为父div的右上角


中间的main通过margin来进行控制扩展


 


 


<!DOCTYPE html PUBLIC “-//W3C//DTD XHTML 1.0 Transitional//EN” “http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd"&gt;
<html xmlns=”http://www.w3.org/1999/xhtml"&gt;
<head>
<meta http-equiv=”Content-Type” content=”text/html; charset=utf-8” />
<title>三列布局</title>
<style type=”text/css”>
body{ margin:0; padding:0; font-size:30px; font-weight:bold;}
div{ line-height:50px;}
.left{ width:200px; height:600px; background:#ccc; position:absolute; left:0; top:0;}
.main{ height:600px;  background:#9CF;margin:0 310px 0 210px; auto;}
.right{ height:600px; width:300px; right:0; top:0; position:absolute; background:#FCC;}
.par{
    margin:0 auto;
    position:relative;
}
</style>
</head>

<body>
    <div class=”par”>
    <div class=”left”>left</div>
    <div class=”main”>设计首页的第一步是设计版面布局。就象传统的报刊杂志编辑一样，我们将网页看作一张报纸，一本杂志来进行排版布局。 虽然动态网页技术的发展使得我们开始趋向于学习场景编剧，但是固定的网页版面设计基础依然是必须学习和掌握的。它们的基本原理是共通的，你可以领会要点，举一反三。</div>
    <div class=”right”>right</div>
    </div>
</body>
</html>

 


再来说一句最常用的混合布局


基本上常用的企业布局都是混合布局


混合布局


定义了一个头部 也就是head


main区域 也就是中间区域


main区域分成了两块 一块是left 一块是right 并且分别浮动


在right中又分成了两块


然后再footer清除浮动


这样一个常用的网站布局就出来了


<!DOCTYPE html PUBLIC “-//W3C//DTD XHTML 1.0 Transitional//EN” “http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd"&gt;
<html xmlns=”http://www.w3.org/1999/xhtml"&gt;
<head>
<meta http-equiv=”Content-Type” content=”text/html; charset=utf-8” />
<title>混合布局</title>
<style>
body{ margin:0; padding:0; font-size:30px; font-weight:bold}
div{ text-align:center; line-height:50px}
.top{ height:100px;background:#9CF}
.head,.main{ width:960px; margin:0 auto;}
.head{ height:100px; background:#F90}
.left{ width:220px; height:600px; background:#ccc; float: left;}
.right{ width:740px; height:600px;background:#FCC; float:right}
.r_sub_left{ width:540px; height:600px; background:#9C3; float:left}
.r_sub_right{ width:200px; height:600px; background:#9FC; float:left;}
.footer{ height:50px; background:#9F9; clear: both;}
</style>
</head>

<body>
<div class=”top”>
    <div class=”head”>head</div>
</div>
<div class=”main”>
    <div class=”left”>left</div>
    <div class=”right”>
        <div class=”r_sub_left”>sub_left
        </div>
        <div class=” r_sub_right”>sub_right
        </div>
    </div>
</div>
<div class=”footer”>footer</div>
</body>
</html>
