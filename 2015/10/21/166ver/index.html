JavaScript:通过索引来创建个选项卡

 


演示页面：http://www.old.com/demo/%E9%80%89%E9%A1%B9%E5%8D%A1.html


学习经验


跟上次那个垂直定位网页特效建立大同小异.


这次主要的代码是


 get_p[i].id=i

上面这句代码给每个 p标签都创建了一个id


每次移动鼠标都会给所有标签添加上display=”none”


 get_menu_ul[this.id].style.display=”block”

然后get_menu_ul


this.id的意思是当前鼠标移入的P标签id是多少 比如id是0


那么对应的ul标签也是0


通过get_p id来建立索引

    function hasClass(ele,cls) {
    return ele.className.match(new RegExp(‘(\s|^)’+cls+’(\s|$)’));
}

function addClass(ele,cls) {
    if (!this.hasClass(ele,cls)) ele.className += “ “+cls;
}

function removeClass(ele,cls) {
    if (hasClass(ele,cls)) {
        var reg = new RegExp(‘(\s|^)’+cls+’(\s|$)’);
        ele.className=ele.className.replace(reg,’ ‘);
    }
}
