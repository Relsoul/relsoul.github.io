jquery:鼠标跟随动画

好吧,来写一个鼠标跟随事件.虽然网上有很多这样的博文..但是. 代码太多 太长不看.自己参考着网上的写了一篇..多动手很容易就理解了


 


本次的任务是:



  1. 获取神器

  2. 拯救世界

  3. 打倒大魔王


真-本次任务



  1. 鼠标跟随的时候读取鼠标移入的那个元素的title属性同时把这个title属性显示出来


 


首先写html


前略-天国的头部君


ok定义了几个P标签


<body>
    <p title=”hello”>test</p>

    <p title=”soul”>test</p>

    <p title=”seeker”>test</p>
</body>


 


二.开始写js文件


引入了jquery(版本>1.8)后先写个$(function(){}) 这样做是为了等网页加载完毕后再执行. 以下的内容就不写这个咯


 


var x=20;
var y=20;
$(‘p’).mouseover(function(event) {
            if(!this.title){return false};
            if($(‘#fllow’)){$(‘#fllow’).remove()}
            this.mytitle=this.title;
            this.title=””;
            var newdiv=”<div id=fllow>”+this.mytitle+”</div>”;
            $(‘body’).append(newdiv);
            $(“#fllow”).stop().animate({
                left: (event.pageX+x)+’px’,
                top:(event.pageY+y)+’px’},
                1000);

        })

以上写了鼠标移入P元素事件,来一条一条分析


var x=20;
var y=20;

x,y是为了后面加一点距离而定义的变量


if(!this.title){return false};
            if($(‘#fllow’)){$(‘#fllow’).remove()}

这两条暂时不管，后面就知道了 先看下面的


 


this.mytitle=this.title;
            this.title=””;

众所周知,浏览器会给默认的title标题一个提示的事件,为了避免这个事件 我们用一个mytitle属性来储存title，同时把title属性设置为空


 


var newdiv=”<div id=fllow>”+this.mytitle+”</div>”;

新建个div,把div的id设置为fllow 同时把mytitle


$(‘body’).append(newdiv);

在body标签中添加上这个div.如果你不想在body中添加上这个标签 你也可以在其他标签里添加


 


在这个时候得说一下fllow CSS设置了


#fllow{
        position: absolute;
        opacity: 1;
        display: block;
    }

设置定位为绝对定位,透明度设置1，类型为块级元素


 


$(“#fllow”).stop().animate({
                left: (event.pageX+x)+’px’,
                top:(event.pageY+y)+’px’},
                1000);

获取id为fllow的标签, .stop()函数是停止正在执行动画的元素, 然后为这个标签添加上动画


event是传递进来的参数, event.pageX获取到鼠标所在的X位置,event.pageY获取到鼠标所在的y位置.


然后动画执行的时间是1000毫秒也就是10秒


这是鼠标移入事件,再来看鼠标移出事件


 


我用的是链式操作，所以前面有个点


.mouseout(function(event) {
            this.title=this.mytitle;
            $(“#fllow”).stop().animate({
                opacity:0,
                display:’none’
            },500);
        })

首先恢复标题内容 this.title=this.mytitle;


然后设置消失动画 透明度从0变为1,隐藏掉fllow元素，动画执行时间是500毫秒


 


ok现在可以来分析上方的判断了


if(!this.title){return false};
            if($(‘#fllow’)){$(‘#fllow’).remove()}

首先,我们隐藏掉fllow标签,但是没有删除掉,也就是说fllow标签还是在页面上的 我们先判断下鼠标移入的标签上有没有title标题,如果没有就直接跳出函数 然后再判断fllow标签是否存在,存在的话删除他


 


现在来分析鼠标在标签上移动事件


 


 


.mousemove(function(event) {
            if(!$(“#fllow”).is(“:animated”)){
                $(“#fllow”).stop().animate({
                left: (event.pageX+x)+’px’,
                top:(event.pageY+y)+’px’}
                ,100);
            }

        });

 


if(!$(“#fllow”).is(“:animated”))

这句话判断 fllow标签没有正在执行动画 是的话则执行函数


$(“#fllow”).stop().animate({
                left: (event.pageX+x)+’px’,
                top:(event.pageY+y)+’px’}
                ,100);

鼠标跟随事件,设置fllow标签的x y为鼠标所在的位置 同时加上一点点距离 动画执行时间是1s


 


 


到此鼠标动画已经写完
