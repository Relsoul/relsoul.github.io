论如何使用jQuery插件flickity设计响应式幻灯片插件效果

发自:http://www.gbtags.com/gb/share/5553.htm

 

最终实现效果如下哈~~

这次给大家推荐一个比较好用的响应式幻灯片,值得一提的是此库不仅仅支持图片的一个幻灯效果,也支持div的幻灯效果,大家可以利用这库实现很多炫酷的效果,具体的看演示吧.

官网: http://flickity.metafizzy.co/

浏览器支持:IE8+,Android 2.3+, iOS Safari 5+, chrome,FF…

快速入门

如何引入. flickity库？




  1. <link rel=“stylesheet” type=“text/css” href=“http://flickity.metafizzy.co/flickity.css“>

  2. <script type=“text/javascript” src=“http://cdn.gbtags.com/jquery/2.1.1/jquery.min.js“></script>

  3. <script type=“text/javascript” src=“http://flickity.metafizzy.co/flickity.pkgd.min.js“></script>




结合jquery来使用该库如下例






  1. $(“#box_img”).flickity({

  2. cellAlign:left

  3. })





不结合jquery来使用该库如下例






  1. var elem = document.querySelector(‘.main-gallery’);

  2. var flkty = new Flickity( elem, {

  3. cellAlign: ‘left’,

  4. contain: true

  5. });


  7. var flkty = new Flickity( ‘.main-gallery’, {

  8. // options

  9. });

详细属性与方法介绍

flickity库属性





  1. 循环滚动

  2. wrapAround: true|false


  4. 自由滚动

  5. freeScroll


  7. 是否显示上下切换条

  8. prevNextButtons: false,


  10. 是否显示小圆点导航

  11. pageDots: false


  13. 自动播放,1500毫秒切换下一图片

  14. autoPlay: 1500 //设置为true默认为3s


  16. 设定幻灯片从第几张开始

  17. initialIndex:1 //设定幻灯片从第几张开始 从0开始计数


  19. 是否启用键盘切换

  20. accessibility: true,//是否启用键盘切换 默认true


  22. 幻灯片高度设置

  23. setGallerySize: true//默认使用最高的图片作为幻灯片的高度,设置为false则使用幻灯片定义高度


  25. 幻灯片播放位置

  26. rightToLeft: true//从右至左播放幻灯片


  28. 预览导航设置

  29. //创建一个新的预览导航幻灯片

  30. $(“.main_box_nav”).flickity({

  31. asNavFor: ‘.main_box’, //指向主幻灯片

  32. pageDots:false,

  33. contain:true

  34. })


  36. /预览导航样式/

  41. /预览导航样式选中状态/

  42. .main_box_nav .nav.is-nav-selected{
