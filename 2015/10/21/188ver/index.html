Html5+Css实现下拉菜单

比较考验Css基础…我这个半吊子快速入门的Css水平还有待提高啊.

代码没啥难度 主要的是一个定位 绝对定位和相对定位.

还有一个选择器

#nav li:hover #nav2{

display: block;

}

//鼠标经过一级菜单时候 设置二级菜单ul为块状

#nav2{

display:none;

}

//二级菜单默认是隐藏的

然后就是细节方面的微调

度娘下载

position：absolute 绝定定位 ,相对其最近的已定位的祖先元素(即是设置了非static的祖先元素)进行定位。如果元素没有已定位的祖先元素，那么它的位置则是相对于最初的包含块（body）。

 

<!doctype html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>HtmlCss实现下拉菜单</title>
    <style type="text/css">
    *{
        margin: 0;
        padding: 0;
    }
    #nav{
        background-color: gray;
        margin: 0 auto;
        width: 600px;
        height: 60px;
        list-style: none;
        position: relative;

    }
    #nav li{
        float: left;
        line-height: 60px;/*设置垂直,也就是行高*/
        text-align: center; /*设置垂直(因为上面设置了行高),和水平居中*/

    }
    #nav li:hover #nav2{
        display: block;
    }
    #nav li a{
         padding: 0 10px;
        /* padding值加在 a标签里是为了防止li宽度加宽 a标签移动时背景颜色覆盖不全*/
        text-decoration: none;
        /*设置a标签下划线为空*/
        color: black;
        /*a标签颜色 黑色*/
        display: block;
        /*设置块状元素的原因是
        鼠标经过a标签的时候
        背景颜色可以覆盖整个a标签块*/
    }
    #nav li a:hover{
        color: #FFF;
        background: #2D2D29;
    }
    #nav2{
        list-style: none;
        position: absolute;
        width: 85px;
        display: none;
    }
    #nav2 li{
        float: none;
        top: 60px;
        left: 0px;
        background: #215A6D;
        margin-top:2px;

    }
    </style>
</head>
<body>
    <ul id="nav">
        <li><a href="#">首页</a></li>
        <li><a href="#">灵魂城堡</a>
            <ul id="nav2">
                <li><a href="#">Soul</a></li>
                <li><a href="#">Seeker</a></li>
            </ul>
        </li>
        <li><a href="#">放荡者茶会</a></li>
        <li><a href="#">记录的地平线</a></li>
    </ul>

</body>
</html>
