JS程序设计-词法重点

JS-词法重点


一:变量标识符的命名要求



  • 以字母,下划线，或者美元符号($)开头

  • 以字母,下划线,美元符号($)和数组组成



二:关键字与保留字


无法使用关键字与保留字来命名


参考地址:http://www.itxueyuan.org/view/6627.html


三:大小写敏感



  • var Aa=0;与var AA=0;是两个变量

  • 关键字比如for也需要大小写分明


四:严格模式



  • 增加js的安全性

  • 提高编译效率


​


五:使用严格模式



  • 全局使用


<script>

“use strict”

var a=0;

</script>



  • 局部使用


function ab(){

“use strict”

var a=0;

}


六:严格模式与标准模式区别



  • 严格模式下不允许隐式声明


function(){

“use strict”

a=0;//a在前面并未声明 ,使用严格模式会报错


}



  • 对象重命名属性


var obj={a:1,b:2,a:3}

obj.a//非严格模式会得出3 严格模式会报错



  • arguments.callee


var i=0;

function ac(){

if(i>10){

return false;

}

i++;

arguments.callee()//指向本身这个函数,严格模式下会报错

}



  • with语句等等~


七:注释



  • 单行注释


//var a=1;

//var b=2;



  • 多行注释


/var a=1;

var b=2;/
